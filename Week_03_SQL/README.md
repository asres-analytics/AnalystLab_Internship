# Week 03 - SQL for Data Analysis

## 📊 Overview
This week focuses on learning SQL for data analysis. The goal is to understand how to query, filter, and analyze structured data from databases.

---

## 🎯 Learning Objectives
- Understand SQL basics
- Write SELECT queries
- Filter data using WHERE
- Use GROUP BY for aggregation
- Learn JOIN operations
- Perform basic data analysis using SQL

---

## 📁 Folder Structure
- scripts/ → SQL query files (.sql)
- notes/ → Learning notes and explanations

---

## 🧠 Skills Gained
- Data querying
- Data filtering
- Aggregation
- Analytical thinking using SQL

---

## 🚀 Goal
By the end of this week, I should be able to analyze datasets using SQL confidently and prepare for real-world business queries.
