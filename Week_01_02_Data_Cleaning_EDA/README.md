## 📌 Project Overview

This project is part of the AnalystLab Africa Data Analytics Internship.
The goal of this project is to clean, process, and analyze raw datasets to make them suitable for analysis and extract meaningful insights.

Two datasets are used in this project:

- Online Retail Dataset
- Netflix Movies & TV Shows Dataset

---

## 🎯 Objectives

- Load and understand raw datasets
- Perform data cleaning and preprocessing
- Handle missing values and duplicates
- Standardize data formats
- Conduct exploratory data analysis (EDA)
- Generate insights and visualizations

---

## 📊 Datasets

### 1. Online Retail Dataset

- Contains transactional data of an online retail store
- Includes customer purchases, product details, and invoice information

### 2. Netflix Dataset

- Contains information about movies and TV shows on Netflix
- Includes title, director, country, release year, and rating

---

## 🧹 Data Cleaning Steps

- Identified missing values
- Removed duplicate records
- Standardized column names
- Converted data types
- Handled inconsistent values
